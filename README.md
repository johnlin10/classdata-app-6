## What's New?

在此新版本中，我們將提供以下新功能：

- 改進了網站結構
- 煥然一新的操作介面
- 更多自定義設定
- 更全面的聊天功能
  - 私人訊息
  - 創建群組
  - 保護隱私（端對端加密訊息）
- 班群貼文
  - 班群貼文頁面
  - 發布貼文
  - 貼文互動
- 個人檔案
  - 瀏覽他人個人檔案
  - 編輯個人檔案
